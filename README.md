# PorterInstall
Installs Porter by @lanewinfield.

##Usage
Copy and Paste the command from below into your Raspberry Pi:
`sudo git clone https://github.com/Shugabuga/PorterInstall.git && cd PorterInstall && sudo sh porter.sh`
